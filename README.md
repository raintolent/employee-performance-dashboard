# Employee Performance Dashboard (Excel)

## Overview
This project demonstrates an Excel-based employee performance dashboard designed to support leadership decision-making.  
The dashboard tracks key performance, workload, and attrition risk indicators across a management hierarchy.

## Business Objective
Enable managers to:
- Monitor team performance at manager and supervisor levels
- Identify attrition and workload risk
- Compare current performance against organizational benchmarks
- Analyze metrics by employee tenure

## Key KPIs
- Average Performance Score
- Average Employee Satisfaction
- Total Projects Handled
- Average Work Hours per Week
- Average Overtime Hours
- Total Sick Days
- Attrition Rate (%)

## Features
- Management hierarchy (Manager / Supervisor)
- Tenure-based cohort filtering
- Benchmark comparison using organizational averages
- Conditional formatting indicators for risk highlighting
- Interactive slicers for dynamic analysis

## Tools Used
- Microsoft Excel
  - PivotTables
  - Conditional Formatting
  - Slicers
  - Dashboard design best practices
